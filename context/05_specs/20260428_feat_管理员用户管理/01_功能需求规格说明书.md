# 功能需求规格说明书

## 1. 概述

### 1.1 功能背景

平台管理员需要跨店铺管理所有用户账号，包括店铺用户和普通用户。管理员可查看所有用户列表、编辑用户基本信息、重置用户密码，确保平台运营秩序。

### 1.2 本功能业务目标

1. 为管理员提供统一的用户列表视图，支持按角色筛选店铺用户/普通用户
2. 支持管理员编辑用户昵称和状态
3. 支持管理员强制重置用户密码

## 3. 页面与功能总览

### 3.1 页面清单

| 序号 | 页面名称 | 页面形式 | 职责说明 | 包含功能 |
|-----|---------|---------|---------|---------|
| 1 | 用户管理列表 | 独立页面 | 展示所有用户列表，支持角色筛选 | 查看用户列表、编辑用户、重置密码 |
| 2 | 编辑用户 | 弹窗对话框 | 编辑用户昵称和状态 | 保存取消 |
| 3 | 重置密码确认 | 弹窗对话框 | 显示重置后的新密码 | 确认关闭 |

### 3.2 页面跳转流程

```
[用户管理菜单] → 用户管理列表页
                      ↓
                点击"编辑" → [编辑用户弹窗] → 保存/取消 → 返回列表
                      ↓
                点击"重置密码" → [重置密码确认弹窗] → 确认关闭 → 返回列表
```

**流程说明：**
1. 管理员从菜单进入用户管理列表页
2. 通过顶部角色筛选条件切换查看「店铺用户」或「普通用户」
3. 点击行内「编辑」按钮打开编辑弹窗
4. 点击行内「重置密码」按钮打开密码确认弹窗

## 4. 页面功能详细说明

### 按钮级别说明（通用）

> 本小节定义按钮级别的通用概念，各页面具体按钮在"功能与按钮"小节描述。

| 级别 | 位置 | 作用域 | 示例 |
|-----|-----|-------|------|
| **页面级按钮** | 页面顶部工具栏 | 针对整个页面或全局操作 | 刷新 |
| **行级按钮** | 列表/表格的每行操作列 | 针对单条数据记录 | 编辑、重置密码 |
| **字段级按钮** | 字段右侧或内部 | 针对单个字段的辅助操作 | 无 |

### 4.1 页面 1：用户管理列表

#### 4.1.1 页面概述

**页面形式：**独立页面

**页面职责：**为管理员提供统一的用户列表视图，支持按角色筛选查看所有店铺用户或普通用户，并提供编辑和密码重置功能。

#### 4.1.2 涉及字段

**【列表页查询字段】**

##### A. 查询字段（列表页专用）

| 字段名称 | 字段类型 | 数据来源 | 校验规则 | 默认值 | 业务含义 |
|---------|---------|---------|---------|--------|--------|
| 角色 | 下拉选择 | 用户输入 | 必填 | 全部 | 筛选用户类型：全部/店铺用户/普通用户 |

##### B. 显示字段

> 列表表格中显示的字段

| 字段名称 | 字段类型 | 数据来源 | 获取时机 | 校验规则 | 默认值 | 业务含义 |
|---------|---------|---------|---------|---------|--------|--------|
| 昵称 | 文本 | 数据库查询 | 页面加载 | 无 | 无 | 用户的显示名称 |
| 所属店铺 | 文本 | 数据库查询 | 页面加载 | 无 | 无 | 用户所属的店铺名称 |
| 积分余额 | 数字 | 数据库查询 | 页面加载 | 无 | 无 | 普通用户专用，显示当前积分余额 |
| 状态 | 标签 | 数据库查询 | 页面加载 | 无 | 无 | 启用/禁用状态 |
| 创建时间 | 日期时间 | 数据库查询 | 页面加载 | 无 | 无 | 账号创建时间 |

**显示字段说明：**
- 店铺用户列表显示：昵称、所属店铺、状态、创建时间
- 普通用户列表显示：昵称、所属店铺、积分余额、状态、创建时间

#### 4.1.3 功能与按钮

**用户可操作功能：**

- **角色筛选**
  - 触发按钮：角色下拉选择器
  - 按钮级别：页面级
  - 按钮位置：列表页顶部工具栏
  - 触发方式：选择选项
  - 权限要求：管理员
  - 加载状态：切换时显示loading
  - 功能说明：选择「店铺用户」显示店铺用户列表，选择「普通用户」显示普通用户列表

- **编辑用户**
  - 触发按钮：编辑
  - 按钮级别：行级
  - 按钮位置：列表操作列
  - 触发方式：点击按钮
  - 权限要求：管理员
  - 加载状态：点击后显示loading，弹窗打开后消失
  - 功能说明：点击后打开「编辑用户」弹窗，可修改用户昵称和状态

- **重置密码**
  - 触发按钮：重置密码
  - 按钮级别：行级
  - 按钮位置：列表操作列
  - 触发方式：点击按钮
  - 权限要求：管理员
  - 加载状态：点击后显示loading，弹窗打开后消失
  - 功能说明：点击后系统随机生成新密码，弹出密码确认框显示新密码，由管理员线下告知用户

**后台自动流程（如有）：**

- **列表自动加载**
  - 触发时机：页面加载时
  - 说明：页面加载时自动查询并展示用户列表，默认显示「全部」角色的用户

#### 4.1.4 业务规则

- **规则1：角色筛选联动**
  - 规则描述：切换角色筛选时，列表数据自动刷新，显示对应角色的用户
  - 触发条件：选择「全部」「店铺用户」或「普通用户」
  - 约束说明：「全部」时同时显示两类用户，通过「角色」列区分

#### 4.1.5 交互逻辑

- 列表排序：默认按创建时间倒序排列
- 分页：支持分页展示，每页20条
- 空状态：无数据时显示「暂无用户」提示

#### 4.1.6 权限要求

> **参见第2.2节"页面访问权限"**

### 4.2 页面 2：编辑用户

#### 4.2.1 页面概述

**页面形式：**弹窗对话框

**页面职责：**编辑用户的基本信息，包括昵称和状态。

#### 4.2.2 涉及字段

| 字段名称 | 字段类型 | 数据来源 | 获取时机 | 校验规则 | 默认值 | 业务含义 |
|---------|---------|---------|---------|---------|--------|--------|
| 昵称 | 文本 | 数据库查询回填 | 弹窗打开时 | 必填，2-50字符 | 原值 | 用户显示名称 |
| 状态 | 下拉选择 | 数据库查询回填 | 弹窗打开时 | 必填 | 原值 | 启用/禁用状态 |

#### 4.2.3 功能与按钮

**用户可操作功能：**

- **保存修改**
  - 触发按钮：保存
  - 按钮级别：页面级
  - 按钮位置：弹窗底部
  - 触发方式：点击按钮
  - 权限要求：管理员
  - 加载状态：点击后显示loading，保存成功后关闭弹窗
  - 功能说明：校验表单后保存用户修改，关闭弹窗并刷新列表

- **取消编辑**
  - 触发按钮：取消
  - 按钮级别：页面级
  - 按钮位置：弹窗底部
  - 触发方式：点击按钮
  - 权限要求：无
  - 加载状态：无
  - 功能说明：关闭弹窗，不保存任何修改

#### 4.2.4 业务规则

- **规则1：状态变更影响**
  - 规则描述：用户被禁用后无法登录系统
  - 触发条件：保存状态为「禁用」
  - 约束说明：无

#### 4.2.5 交互逻辑

- 弹窗打开：点击编辑按钮后弹出表单弹窗，表单自动填充当前用户数据
- 弹窗关闭：保存成功或点击取消/遮罩时关闭弹窗

#### 4.2.6 权限要求

> **参见第2.2节"页面访问权限"**

---

### 4.3 页面 3：重置密码确认

#### 4.3.1 页面概述

**页面形式：**弹窗对话框

**页面职责：**显示系统随机生成的新密码，供管理员线下告知用户。

#### 4.3.2 涉及字段

| 字段名称 | 字段类型 | 数据来源 | 获取时机 | 校验规则 | 默认值 | 业务含义 |
|---------|---------|---------|---------|---------|--------|--------|
| 新密码 | 文本（只读） | 系统生成 | 弹窗打开时 | 无 | 随机生成 | 系统随机生成的8位密码 |

#### 4.3.3 功能与按钮

**用户可操作功能：**

- **确认关闭**
  - 触发按钮：确认
  - 按钮级别：页面级
  - 按钮位置：弹窗底部
  - 触发方式：点击按钮
  - 权限要求：管理员
  - 加载状态：无
  - 功能说明：关闭弹窗，管理员需自行线下告知用户新密码

#### 4.3.4 业务规则

- **规则1：密码生成规则**
  - 规则描述：系统随机生成8位密码，包含数字和字母
  - 触发条件：点击重置密码按钮
  - 约束说明：密码必须告知用户后生效，弹窗关闭后密码无法找回

#### 4.3.5 交互逻辑

- 弹窗打开：点击重置密码按钮后弹出，显示系统生成的随机密码
- 密码展示：密码明文显示，提供复制按钮

#### 4.3.6 权限要求

> **参见第2.2节"页面访问权限"**
